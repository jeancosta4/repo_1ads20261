# 📘 Exemplo de Documento em Markdown

## 🧾 Introdução
Este é um exemplo simples de como estruturar um texto utilizando **Markdown**.  
O Markdown é muito utilizado para documentação, README de projetos e anotações.

---

## 🎯 Objetivo
Demonstrar os principais recursos do Markdown, como:

- Títulos
- Listas
- Negrito e itálico
- Links
- Código

---

## ✍️ Formatação de Texto

Você pode escrever em **negrito**, *itálico* ou até mesmo ***negrito e itálico juntos***.

Também é possível destacar trechos de código inline, como:  
`print("Olá, mundo!")`
